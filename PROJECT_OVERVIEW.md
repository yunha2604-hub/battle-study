# 스쿨배틀 (SchoolBattle Arena) — 프로젝트 개요 및 기능 설명

## 프로젝트 소개

**스쿨배틀**은 중학교 3학년 창업 아이디어 발표를 위해 직접 개발한 웹 시제품입니다.
"공부를 게임처럼"이라는 컨셉 아래, 리그 오브 레전드(LoL) 스타일의 UI로 학생들이 학교 명예를 걸고 1대1 실시간 퀴즈 배틀을 펼치는 서비스입니다. 선생님을 위한 AI 출제·생기부 세특 자동 생성 기능도 함께 제공합니다.

---

## 화면별 기능 설명

### 1. 랜딩 페이지 (LandingPage)

**파일:** `src/components/LandingPage.tsx`

회원가입 없이 닉네임과 학교 입력만으로 즉시 입장하는 진입 화면입니다.

| 기능 | 설명 |
|------|------|
| 닉네임 입력 | 최대 12자, 게임 내 표시 이름 |
| 학교 검색 자동완성 | 입력 시 중학교 목록 드롭다운 필터링 |
| 즉시 배틀 시작 | 입력 후 로비를 거치지 않고 첫 매칭 자동 진입 (온보딩 최소 마찰) |
| 교사 포털 진입 | 우측 상단 버튼으로 선생님 대시보드 이동 |
| 배경 애니메이션 | 글로우 이펙트, 파티클 스파클 |

**핵심 설계 의도:** 가입 절차가 귀찮아서 이탈하는 학생을 막기 위해, 닉네임과 학교만 입력하면 1초 만에 배틀이 시작되도록 설계.

### 2. 로비 (Lobby)

**파일:** `src/components/Lobby.tsx`

배틀 대기 공간이자 게임의 허브 화면입니다. 세 개의 탭으로 구성됩니다.

#### 탭 1: 홈 / 매칭 (ARENA)

| 기능 | 설명 |
|------|------|
| 내 프로필 카드 | 닉네임, 학교, 현재 티어, LP 바 시각화 |
| 티어 시스템 | 아이언 / 브론즈 / 실버 / 골드 / 다이아몬드 5단계 (각 별명 포함) |
| 에너지(번개) 시스템 | 최대 5개, 매칭 1회당 1개 소비. 0개 시 매칭 불가 → 오답 던전에서 충전 유도 |
| 개인 전적 | 승·패·승률, 연승 보너스 LP 알림 |
| 오늘의 현상금 퀘스트 | 일일 미션 3개 (삼각함수 3회 처치 / 5초컷 2승 / 복수전 1회), XP 보상 |
| 전국 학교 랭킹 | 학교별 참여 인원·누적 LP 순위표, 내 학교 자동 하이라이트 |
| 자동 매칭 | 과목(국어·영어·수학) 선택 후 "매칭 시작" → 3단계 탐색 애니메이션 후 배틀 진입 |
| 방 만들기 | 6자리 랜덤 PIN 코드 생성 → 친구에게 공유 |
| 코드로 입장 | 받은 PIN 입력 후 커스텀 방 대기실 진입 |
| 이벤트 배너 | 학원 스폰서 학교 대항전 이벤트 공지·참가 (예: 동탄고 vs 반송고) |
| 티어 디자인 테스트 도구 | 설정 버튼으로 펼치는 숨김 패널, 티어·LP를 실시간 조정해 UI 확인 |
| 접속자 수 표시 | "4,821명 접속 중" 실시간 효과 (데모 수치) |

#### 탭 2: 전적 분석 — OP.GG 탭 (ANALYTICS)

**파일:** `src/components/PlayerAnalytics.tsx`

OP.GG 스타일의 개인 퀴즈 전적 분석 화면입니다.

| 기능 | 설명 |
|------|------|
| 과목별 정답률 차트 | 국어·영어·수학 분야 실력 시각화 |
| 배틀 히스토리 | 최근 경기 결과·상대 정보·LP 변동 목록 |
| 취약 유형 분석 | 자주 틀린 문제 카테고리 집계 |

#### 탭 3: 오답 던전 — Shadow Raid (SHADOW_RAID)

**파일:** `src/components/ShadowRaid.tsx`

틀린 문제를 다시 풀어 에너지를 충전하는 복습 모드입니다.

| 기능 | 설명 |
|------|------|
| 오답 몬스터 사냥 | 이전 배틀에서 틀린 문제가 몬스터로 등장 |
| 에너지 충전 | 오답 문제를 맞추면 번개(에너지) 회복 |
| 던전 탐색 | 자기주도적 복습 유도 구조 |

### 3. 배틀 아레나 (BattleArena)

**파일:** `src/components/BattleArena.tsx`

스쿨배틀의 핵심 화면으로, 두 가지 모드(일반 배틀 / 수행평가 모드)를 하나의 컴포넌트로 처리합니다.

#### 3-A. 일반 배틀 모드 (다크 테마)

| 기능 | 설명 |
|------|------|
| HP 바 | 내 HP / 상대 HP 애니메이션 바, 수치에 따라 초록→노랑→빨강 색상 변화 |
| 라운드 진행 | 과목별 2~3문제, 라운드 표시 및 카테고리 뱃지 |
| 개별 제한 시간 | 문제마다 다른 타임리밋 (예: 어휘 10초, 보스 배틀 90초), 카운트다운 바 |
| 타임어택 크리티컬 히트 | 제한 시간의 30% 이내 정답 → 데미지 2배 (68 vs 34), CRITICAL HIT x2 팝업 |
| 심리 압박 시뮬레이터 | 상대가 먼저 풀었다는 메시지가 뜨고 남은 시간이 10초로 단축, 화면 테두리가 붉게 맥박 |
| 공격 이펙트 | 정답 시 칼 슬래시 애니메이션 + 상대 쉐이크, 오답 시 레드 비네트 + 내 화면 쉐이크 |
| 시간 초과 처리 | 제한 시간 경과 시 자동 오답 처리 (HP -34) |

#### 3-B. 학교 대항전 모드 (isTeamBattle)

| 기능 | 설명 |
|------|------|
| 공유 체력 줄다리기 바 | Team A / Team B 공유 HP가 하나의 가로 바에서 좌우로 당겨지는 시각화 |
| 실시간 이벤트 알림 토스트 | 화면 우하단에 팀원들의 공격/오답 결과가 실시간 스크롤 (시뮬레이션) |
| 팀 기여 반영 | 정답 시 Team B HP -25, 오답 시 Team A HP -20 즉시 반영 |

#### 3-C. 수행평가 엄근진 모드 (isStrictAssessment)

수행평가 시 선생님이 활성화하는 시험 전용 화면입니다.

| 기능 | 설명 |
|------|------|
| 흰색 시험지 테마 | 다크 배경→흰색 배경으로 전환, 화려한 이펙트 전체 제거 |
| 닉네임 익명화 | 학생 닉네임이 "Student A (본인)" / "Student B"로 자동 블라인드 처리 |
| 이탈 방지 경고 배너 | 상단에 빨간 경고 배너 "화면 이탈 시 0점 처리" 상시 표시 (애니 펄스) |
| 크리티컬 히트 비활성화 | 심리 압박·이펙트 등 게임 요소 전면 제거, 순수 시험 환경 |

### 4. 결과 페이지 (ResultPage)

**파일:** `src/components/ResultPage.tsx`

경기 종료 후 승패 결과, LP 변동, AI 오답 분석을 보여주는 화면입니다.

| 기능 | 설명 |
|------|------|
| VICTORY / DEFEAT 타이틀 | 승패에 따른 그라디언트 텍스트 + 글로우 배경 |
| 정답 수 · 최종 HP 요약 | 경기 주요 지표 한 줄 요약 뱃지 |
| LP 카운팅 애니메이션 | 승리 +20LP / 패배 -15LP가 숫자가 하나씩 올라가며 시각화 |
| 티어 승격 연출 | LP가 100을 돌파하면 전체 화면 오버레이로 "TIER PROMOTED!" 연출 |
| AI 오답 분석 피드백 | 문제별 카드 아코디언 — 내 선택·정답·AI 튜터 해설 |
| 카카오 계정 연동 팝업 | 첫 경기 후 기록 저장 유도 (성장 해킹 리텐션 팝업, 카카오 로그인 UI 포함) |

### 5. 선생님 대시보드 (TeacherDashboard)

**파일:** `src/components/TeacherDashboard.tsx`

선생님 전용 흰색 B2B 화면으로, AI 출제와 생기부 세특 작성을 지원합니다.

#### 5-A. 출제 배틀 모드 제어

| 기능 | 설명 |
|------|------|
| 일반 배틀 / 수행평가 모드 토글 | 스위치 한 번으로 학생 화면 전환 (일반=게임 이펙트 / 수행평가=시험지 모드) |
| 아레나 테스트 실행 버튼 | 현재 모드로 배틀 아레나 즉시 미리보기 |

#### 5-B. AI 1초 문항 출제 엔진

| 기능 | 설명 |
|------|------|
| 평가 범위 설정 | 출판사(미래엔/천재교육/비상/EBS/YBM) → 학년·과목 → 단원·목차 3단 연계 드롭다운 |
| 성취기준 자동 매핑 | 선택한 단원에 따라 교육과정 코드·성취기준명 자동 표시 (예: [12영01-03] 목적에 맞는 어휘 선택) |
| 교과서 지문 입력 | 텍스트 복붙 입력란 |
| AI 자동 문제 생성 | 버튼 클릭 → 1.5초 후 국어·영어·수학 다지선다 3문항 자동 생성 |
| 난이도 조정 | 생성된 문항별 난이도 드롭다운 (하 / 중 / 수능형 킬러) |

#### 5-C. AI 세특 분석 리포트

| 기능 | 설명 |
|------|------|
| 학생 선택 | 퀴즈 전적이 있는 학생 목록 버튼 |
| 개인 통계 | 평균 정답률 + 평균 풀이 반응속도 |
| 생기부 세특 추천 문구 | 선택한 교육과정 범위·성취기준과 학생 퀴즈 기록을 조합한 생기부 단락 자동 생성 |
| 클립보드 복사 | "문구 복사하기" 버튼 → NEIS 생기부 시스템에 바로 붙여넣기 가능 |

## 게임 핵심 규칙 요약

| 규칙 | 내용 |
|------|------|
| HP 시스템 | 양측 시작 HP 100. 정답 시 상대 HP -34 (크리티컬 시 -68), 오답 시 내 HP -34 |
| 크리티컬 히트 | 제한 시간의 30% 이내 정답 → 데미지 2배 |
| 타임아웃 | 제한 시간 경과 = 오답 처리 |
| LP 변동 | 승리 +20 LP / 패배 -15 LP |
| 티어 승격 | LP 100 돌파 시 다음 티어 진급 (아이언→브론즈→실버→골드→다이아몬드) |
| 에너지 | 자동 매칭 1회 = 번개 1개 소모. 0개 시 오답 던전 완료 후 충전 |

## 보고서 ↔ 구현 대응표

| 보고서 기술 내용 | 구현된 기능 |
|-----------------|-------------|
| 닉네임+학교만으로 1초 입장 | `handleJoin` → 로비 없이 즉시 `BATTLE` 뷰 전환 |
| LoL 스타일 다크모드 | `BattleArena` 다크 테마, 슬래시·크리티컬 이펙트 |
| 타이머 30% 이내 정답 = 공격력 2배 | `isCritical = timeTaken <= timeLimit * 0.3` |
| 반 대항전 줄다리기 게이지 | `isTeamBattle` 모드, 공유 HP 스프링 애니메이션 바 |
| 실시간 공격 알림 | 팀 배틀 이벤트 토스트 (우하단 슬라이드인) |
| AI 교과서 3초 문제 생성 | `TeacherDashboard` AI 출제 엔진 (시제품: 1.5초 시뮬레이션) |
| 생기부 세특 자동 문구 | `getDynamicReport()` — 교육과정 범위 + 학생 통계 조합 생성 |
| 수행평가 화면 모드 전환 | `isStrictAssessment` — 흰색 시험지, 닉네임 블라인드, 이펙트 제거 |
| 부정행위 방지 경고 | 수행평가 모드 상단 빨간 경고 배너 상시 표시 |
| 다른 창 이탈 경고 | 수행평가 배너 "화면 이탈 시 0점 처리" 문구 (감지 로직 연동 예정) |
